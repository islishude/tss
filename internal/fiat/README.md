# fiat-crypto generated arithmetic

This directory contains Go arithmetic generated by fiat-crypto for the curve
fields used by this module.

The generated packages are:

- `secp256k1field`: secp256k1 base field `2^256 - 2^32 - 977`
- `secp256k1scalar`: secp256k1 subgroup order
- `ed25519scalar`: Ed25519/FROST subgroup order

Regenerate with Docker from the repository root:

```sh
./generate.sh
```
